Lenguajes Script (JavaScript, Ajax, JQuery, Json)

Descripción

Los lenguajes de script son esenciales para la interactividad en la web. JavaScript es el núcleo, permitiendo manipular el DOM y responder a eventos. Ajax facilita la comunicación asíncrona con servidores, jQuery simplifica tareas comunes, y JSON es un formato ligero para intercambio de datos.

Características

  • JavaScript: Lenguaje dinámico, soporta programación funcional y orientada a objetos.
  • Ajax: Permite actualizar partes de una página sin recargarla.
  • jQuery: Biblioteca que simplifica la manipulación del DOM y eventos.
  • JSON: Formato ligero, legible y compatible con la mayoría de lenguajes.
  • Cross-Browser: Alta compatibilidad con navegadores modernos.

Tipos

  • JavaScript Vanilla: JavaScript puro sin bibliotecas.
  • Frameworks JS: Como React, Angular, y Vue.js para aplicaciones complejas.
  • Ajax Tradicional: Usa XMLHttpRequest.
  • Ajax Moderno: Usa Fetch API para solicitudes HTTP.
  • JSON Básico: Estructura de datos simple.
  • JSON-LD: JSON para datos enlazados y SEO.
